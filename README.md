## 기능
### 입출력
- [ ] 로또 구입 금액을 입력
- [ ] 당첨 번호 입력
- [ ] 보너스 번호 입력
- [ ] 당첨 내역 출력
- [ ] 수익률 소수점 둘째 자리에서 반올림하여 출력
- [ ] 예외 발생시 에러 문구 출력 후 재입력

### 검증
- [ ] 구입 금액이 1,000원으로 나누어 떨어지는지 검증
- [ ] 당첨 번호가 1~45사이의 정수인지 검증


### 로또번호
- [ ] 로또 번호는 6개의 숫자를 가짐
- [ ] 랜덤으로 중복없이 생성
- [ ] 당첨번호와 보너스 번호를 입력받아서 등수를 반환

- [ ] 수익률 계산
